# Mandelbrot

Application to visualize the fractal of mandelbrot and julia.

## Windows

```sh
git clone https://github.com/Dashstrom/Mandelbrot.git
py setup.py exe
.\dist\Mandelbrot.exe
```

## Linux

```sh
git clone https://github.com/Dashstrom/Mandelbrot.git
pip install .
python3 -m mandelbrot
```
